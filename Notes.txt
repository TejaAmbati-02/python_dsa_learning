Collections:
from collections import deque, counter, defaultdict, ordereddict, namedtuple


Deque :  (append, appendleft, pop, popleft, extend, extendleft, rotate, len) -> Operations at front and end
counter :  (c[2], most_common_elements, update(add elements to the counter), subtract, +, -, &, ,) -> To maintain counter 
defaultdict : Never Raises Key Error, Creates default value -> Dict Will not give key error if key is not present
ordereddict : in, move_to_end, popitem -> To maintain insertion order, Throw key error if not found
namedtuple : Tuples with named fields (immutable) -> Immutable values but helps in readibility
